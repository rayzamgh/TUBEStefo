\doxysubsection{Detailed Description}
Class yang menampung semua objek yang ada di game. 

Class ini dibuat dengan tujuan menyimpan semua objek yang ada di game dan melakukan manipulasi terhadap objek tersebut. Class ini diimplementasikan menggunakkan Singleton Pattern yaitu hanya ada 1 objek untuk tiap program
